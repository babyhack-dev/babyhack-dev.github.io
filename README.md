# VoiceIME

> 让输入从“说出来”到“发出去”一步完成。  
> 一款面向 macOS 的 AI 输入助手，集成语音转录、AI 优化、智能回复与截图 OCR。

## 产品简介

VoiceIME 将多个高频输入能力整合进菜单栏工作流，帮助用户更高效地完成输入任务。

## 核心能力

- **语音转录**：按住快捷键录音，松开自动转文字
- **AI 优化**：将原始表达整理成更自然、更清晰的文本
- **智能回复**：多角色并发生成回复建议，适合翻译、润色与语气切换
- **截图 OCR**：从截图、网页、图片和文档中提取文本并继续处理
- **菜单栏常驻**：通过快捷键快速唤起核心能力
- **灵活接入**：支持第三方兼容 API，也支持在适用场景下直接复用 ChatGPT OAuth

## 快捷键

| 功能 | 默认快捷键 |
|------|------------|
| 开始录音 | `⌘⇧O` |
| 智能回复 | `⌃⇧R` |
| 截图 OCR | `⌘⇧P` |

## 下载与安装

仓库仅提供 ZIP 安装包下载，压缩包根目录包含：

- `install.sh`
- `VoiceIME.dmg`

### 安装方式一：Terminal 快速安装

1. 下载并解压 ZIP
2. 进入目录后执行 `./install.sh`
3. 按提示完成安装与系统权限授权

### 安装方式二：ZIP 内 DMG 图形安装

1. 下载并解压 ZIP
2. 双击打开 ZIP 内的 `VoiceIME.dmg`，拖入 Applications
3. 首次启动若看到“Apple could not verify 'VoiceIME'...”提示，请前往 **系统设置 → 隐私与安全性**
4. 点击 **Open Anyway** 后再次打开应用即可
