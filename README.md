# VoiceIME

> 面向 macOS 的智能语音输入法。  
> 集成语音输入、智能回复、智能发声、智能 OCR 与智能变音，帮助用户在桌面场景中更高效地完成输入、理解、生成与表达。

## 产品定位

VoiceIME 以菜单栏与全局快捷键为核心入口，将高频输入与表达能力整合为一套原生 macOS 工作流。用户可以通过语音完成输入，通过 AI 生成回复，通过 OCR 获取屏幕内容，并通过智能发声与智能变音完成语音输出。

## 核心功能

### 语音输入法

- 快捷键启动录音，说话后自动生成文本
- 支持 HTTP provider 与本地 SenseVoice 识别方案
- 支持录音来源、文本润色、注入方式等配置
- 适合聊天、写作、记录、文档输入等高频场景

### 智能回复

- 基于选中文本或当前输入生成回复内容
- 支持回复角色、多角色并发、翻译与润色
- 支持模型、Token、注入方式等参数配置
- 适合即时沟通、邮件回复、客服话术与双语表达

### 智能发声

- 将文本转换为自然语音内容
- 支持 Edge TTS、VoxCPM Remote 与本地发声链路
- 支持声音设计、试听文本、播放设备与虚拟麦克风链路
- 适合朗读、演示、内容创作与语音输出

### 智能 OCR

- 识别截图、网页、图片与文档中的文字
- 支持二维码识别与信息提取
- OCR 结果可继续用于复制、编辑、回复或发声
- 适合截图取字、资料整理、链接识别与桌面信息处理

### 智能变音

- 为变音场景独立配置语音识别、角色与发声 provider
- 支持按住说话、松开处理的完整变音链路
- 支持本地与在线发声方案组合
- 适合直播、会议、演示与内容创作

## 角色系统

VoiceIME 提供发言角色与回复角色两类提示词配置，用于管理不同场景下的 AI 表达方式。

- **发言角色**：用于语音输入后的文本润色与表达优化
- **回复角色**：用于智能回复、翻译、沟通与场景化回答
- 支持启用、编辑、排序、新建与恢复默认模板

## 本地与在线能力

VoiceIME 支持 Local 与 HTTP 两类 provider。用户可以根据功能需求与部署偏好选择对应方案。

- 语音输入支持 HTTP 与本地 SenseVoice
- 智能回复支持复用 ChatGPT OAuth 登录态与模型配置
- 智能发声支持 HTTP provider、VoxCPM Remote 与本地链路
- 智能变音支持独立的识别与发声配置

## 下载与安装

当前分发包提供 ZIP 与 DMG 两种安装路径。ZIP 压缩包内包含：

- `install.sh`
- `VoiceIME.dmg`

### Terminal 快速安装

1. 下载并解压发布包
2. 进入解压目录
3. 执行 `./install.sh`
4. 按照系统提示完成权限授权

### DMG 图形安装

1. 下载并解压发布包
2. 打开 ZIP 内的 `VoiceIME.dmg`
3. 将 VoiceIME 拖入 Applications
4. 首次启动时按照 macOS 安全提示完成打开授权

## 常用权限

VoiceIME 的核心能力会根据使用场景请求 macOS 系统权限：

- 麦克风：用于语音输入、智能变音与参考音频录制
- 辅助功能：用于将生成结果注入到当前应用
- 屏幕录制：用于截图 OCR 与屏幕内容识别
