Capabilities

围绕输入与输出的完整能力体系。

VoiceIME 不是单一功能应用，而是一组可以协同工作的原生桌面能力。

Voice Input

语音输入

支持 HTTP 与本地 STT 路线，围绕录音来源、运行时、模型与注入方式形成完整配置面。

Smart Reply

智能回复

支持模型配置、角色复用与多选并发生成，适合沟通、翻译、整理与结构化回复。

Smart Speech

智能发声

统一在线与本地发声 provider，支持试听、参考音频、录音来源与虚拟麦克风链路。

Smart OCR

智能 OCR

将截图取字与二维码识别纳入统一工作流，继续进入后续编辑、回复或发声步骤。

Prompt System

角色系统

发言角色与回复角色分工明确，支持排序、切换、模板恢复与长期维护。

Toolbox

工具魔盒

提供轻量原生工具能力，当前优先提供二维码生成与预览，后续可继续扩展。

Designed for native macOS

以原生桌面体验为前提，而不是把网页操作搬进应用。

VoiceIME 的设计重点，是让菜单栏、快捷键、设置中心、历史系统与角色系统形成稳定的原生工作方式。 这意味着它既适合快速触发，也适合长期配置与持续使用。

FAQ

常见问题

以下问题通常有助于快速判断 VoiceIME 是否符合你的使用方式。

VoiceIME 适合哪些场景？

适合聊天沟通、邮件处理、文档整理、截图取字、翻译润色、朗读与辅助表达等需要频繁切换输入方式的桌面场景。

它只支持在线模型吗？

不是。VoiceIME 同时支持在线服务与本地运行时，可按具体能力选择 HTTP provider 或本地路线。

产品预览是否对应真实功能界面？

是。官网中的预览模块基于当前产品功能结构与真实页面语义组织，便于你在下载前直观看到产品形态。

首次安装后需要做什么？

完成安装后，根据系统提示授权录音、辅助功能、屏幕录制等必要权限，即可开始使用主要能力。
