What’s new

v2.9.8 更新

这次不是单点修补，而是把菜单栏主入口、设置页命名、历史页名称、快捷键与参考录音交互整体收口到同一套心智模型里。

Menu Bar

一级动作只保留四个高频入口

主操作顺序调整为「语音输入 → 智能回复 → 智能发声 → 智能OCR」，快速启动器与设置类入口下沉到次级区域，减少竞争注意力。

Naming

命名统一到用户能立刻理解的术语

语音转录改为语音输入，智能发音改为智能发声；转录历史改为语音历史，发音历史改为发声历史，减少新旧词并存带来的记忆负担。

Hotkeys

默认快捷键重新映射，更贴合首字母

智能回复保持 ⌃⌘U；语音输入调整为 ⌃⌘I；智能发声调整为 ⌃⌘O；智能OCR 与快速启动器继续使用 ⌃⌘P、⌃⌘L。

Recording UX

参考录音来源放到按钮上方并整块高亮

Local / VoxCPM 参考录音来源已并入参考音频素材卡片；准备录音和录音中都会高亮来源区，并明确告诉用户“当前录音对象”究竟是麦克风还是系统声音。

Core capabilities

核心能力

从输入获取到文本处理，再到输出和追溯，VoiceIME 把高频动作整合成一套可持续使用的桌面输入层。

Voice Input

语音输入

按住快捷键即可录音，松开后继续完成转录、AI 润色与文本注入。支持 HTTP STT，也支持本地 Local / SenseVoice 路线。

Smart Reply

智能回复

多角色并发生成回复，可快速对比不同语气、翻译版本或沟通身份，适合微信、邮件、工单与文档整理。

Smart Speech

智能发声

支持 Edge TTS 与 Local / VoxCPM，既能直接朗读选中文本，也能基于参考音频做克隆与自定义声音输出。

Smart OCR

智能OCR

支持截图识别文字与二维码。网页、图片、PDF、验证码、群二维码等场景都能直接进入后续编辑与处理流程。

Quick Launcher

快速启动器

统一搜索面板默认展示工具魔盒、历史记录、设置面板与关于更新，同时保留新旧命名别名，升级后也能快速搜到入口。

History

历史记录

语音历史、回复历史、发声历史与 OCR 历史分组清晰，方便回看原始输入、生成结果和输出轨迹。

Consistency matters

从 Menu Bar 到设置页，这一轮把体验真正打通了。

不仅仅改文字，而是把动作顺序、命名、搜索别名、历史入口与提示反馈统一到一条更自然的使用路径上。

主入口收口

顶部只保留四个最常用动作

  1. 语音输入最常用入口，继续保留按住录音主链路
  2. 智能回复一键生成多角色回复，适合即时沟通
  3. 智能发声短按直接发声，长按可选择发声模式
  4. 智能OCR截图文字 / 二维码继续进入处理流程

快速启动器、快捷键、设置与关于更新不再和核心动作并列占据主注意力，但仍然触手可达。

FAQ

常见问题

如果你打算把 VoiceIME 作为日常输入层，下面几项通常是最先关心的问题。

VoiceIME 适合哪些场景？

适合聊天沟通、邮件撰写、工单回复、文档整理、翻译润色，以及截图取字后的继续编辑与朗读。

2.9.8 这次最值得关注的变化是什么？

核心是统一心智：Menu Bar 顶部只保留四个高频动作，设置页和历史页命名同步统一，快捷键默认映射更直觉，参考录音来源也终于能在录制时明确提示当前对象。

智能发声支持哪些路线？

既支持 HTTP 路线（如 Edge TTS），也支持 Local / VoxCPM。本地模式下可以加载参考音频、做极致克隆，并直接用于试听或朗读结果。

快捷键修改后，旧用户还能继续用吗？

应用内已经处理旧默认值迁移逻辑；同时快速启动器和相关入口也兼容新旧命名，升级后不需要重新摸索整个路径。

首次安装建议怎么做？

建议优先下载 ZIP：既可以直接运行 ./install.sh，也可以打开 ZIP 内的 VoiceIME.dmg。完成安装后，再按权限提示授予录音、辅助功能、自动化和截图相关能力。
